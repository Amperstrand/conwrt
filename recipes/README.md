# Recipes

Vendor- and model-specific knowledge for conwrt.

## Layout

recipes/<vendor>/<model>/notes.md

Example: recipes/linksys/whw03-v2/notes.md

Recipes contain:
- Known-safe read-only API endpoints
- Firmware version history
- OpenWrt Table of Hardware links
- Known gotchas
- Session-validated timing, boot, and recovery notes when available
- Traffic indicators that distinguish "still booting", "ready for stock flash", and "already in OpenWrt"

## Contributing

Add a recipe directory when you have validated information for a specific device.
Do NOT include unverified information.
